This is a 3-key macropad built for Hack Club Hackpad

Features
3 vertical keys
Snipping Tool shortcut
Opens browser (Google)
Spotify play/pause
Hardware
Microcontroller: Seeed Studio XIAO RP2040
Custom PCB (KiCad)
3 mechanical switches
Firmware

Built using KMK Firmware. My firmware is in my code.py folder so when I have built my hackpad I can use KMK firmware with this code

Keymap

Top key: Snipping Tool
Middle key: Open browser
Bottom key: Play/Pause music

BOM (Bill of Materials)

| Part | Quantity | Notes |
|------|----------|-------|
| Seeed XIAO RP2040 | 1 | Main microcontroller |
| 1N4148 Diodes | 20 | Through-hole, for key matrix |
| MX-Style Switches | 16 | Mechanical key switches |
| EC11 Rotary Encoders | 2 | For scroll/volume control |
| 0.91" OLED Display | 1 | Pin order: GND-VCC-SCL-SDA |
| SK6812 MINI-E LEDs | 16 | RGB underglow/per key lighting |
| DSA Keycaps (white blank) | 16 | Fits MX switches |
| M3x16mm Screws | 4 | For case assembly |
| M3x5x4mm Heatset Inserts | 4 | For case assembly |
| 3D Printed Case | 1 | Printed in PETG or PLA+ |
